**ACS-1111 Bank Account**
BankAccount class that can do basic banking operations. Accounts created with attributes: full name, bank account number (unique 8 digit), balance (initial value of 0). It has basic operations of deposit, withdraw, interest (0.83%), and print statement, with methods.

# Languages: python | Libraries: random

# Known Issues:
 - No safeguards for entering wrong type of data, specifically amount params
 - Generating unique 8 digits very inefficient
